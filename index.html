Friendly and compliant implementations of
OpenADR 2.0b and OpenADR 3.0

OpenLEADR is a project that is part of the LF Energy Foundation. It offers a fully featured and compliant OpenADR 2.0b implementation for Python 3 and an OpenADR 3.0 implementation written in Rust.

Source code on GitHub: OpenADR 2.0b, OpenADR 3.0.

OpenADR 2.0b documentation: /docs/.
